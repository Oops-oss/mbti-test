# MBTI 人格测试

一个精美的 MBTI 人格测试网页应用，帮助你发现自己的真实性格类型。

## 功能特点

- 📝 **93道科学题目** - 深入分析你的性格特点
- ⚡ **快速测试** - 约5分钟完成
- 🎯 **16种人格类型** - 详细的 MBTI 人格解读
- 🎨 **精美界面** - 现代设计，流畅体验

## MBTI 人格类型

| 字母 | 含义 |
|------|------|
| I / E | 内向 / 外向 |
| S / N | 感觉 / 直觉 |
| T / F | 思考 / 情感 |
| J / P | 判断 / 知觉 |

## 技术栈

- HTML5 + CSS3
- Vanilla JavaScript
- 响应式设计

## 快速部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Oops-oss/mbti-test)

## 本地运行

```bash
# 克隆项目
git clone https://github.com/Oops-oss/mbti-test.git

# 进入目录
cd mbti-test

# 使用任意 HTTP 服务器运行
# Python
python -m http.server 8000

# 或者 Node.js
npx http-server -p 8000
```

然后访问 http://localhost:8000
